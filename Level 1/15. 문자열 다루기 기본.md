## 문제설명

문자열 s의 길이가 4 혹은 6이고, 숫자로만 구성돼있는지 확인해주는 함수, solution을 완성하세요. 예를 들어 s가 "a234"이면 False를 리턴하고 "1234"라면 True를 리턴하면 됩니다.

**제한사항**

- `s`는 길이 1 이상, 길이 8 이하인 문자열입니다.
- `s`는 영문 알파벳 대소문자 또는 0부터 9까지 숫자로 이루어져 있습니다.

**입출력 예**

| s      | return |
| ------ | ------ |
| "a234" | false  |
| "1234" | true   |

## 풀이

```js
function solution(s) {
  if (s === '0000' || s === '000000') return true;
  return (
    !!Number(s) &&
    Number(s) > 0 &&
    !s.split('').includes('e') &&
    (s.length === 6 || s.length === 4)
  );
}
// 제일 오래 걸린 케이스(0.09ms, 33.5MB)
// 제일 빠른 케이스(0.04ms, 33.4MB)
```

### 설명

- '0000'과 '000000'을 생각 못함. 어떻게 풀어야 될지 감이 안와 일단 하드코딩으로 처리
- 숫자면 true
- 양수면 true
- e가 포함되지 않으면 true
- 문자열 길이가 6이나 4여야 true

## 다른 사람의 풀이

```js
function solution(s) {
  var regex = /^\d{6}$|^\d{4}$/;

  return regex.test(s);
}
// 제일 오래 걸린 케이스(0.10ms, 33.4MB)
// 제일 빠른 케이스(0.03ms, 33.4MB)
```

### 설명

- 정규식
- 숫자 6자리 또는 4자리
